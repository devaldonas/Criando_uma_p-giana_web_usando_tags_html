História da Web

(voltar)

É preciso entender um pouco da história do computador, e como a internet surgiu, para que possamos entender o que acontece quando acessamos uma página web ou utilizamos um aplicativo.

Os primeiros computadores, chamados da primeira geração, surgiram entre a década de 1940 e o final dos anos 1950. Eles funcionavam com válvulas eletrônicas e circuitos.

Eram enormes e pesados,cConsumiam muita energia,eram bem lentos, calculando em milésimos de segundo e as válvulas quebravam com frequência

Os computadores segunda geração (1959-1964), ficaram mais rápidos, menores e baratos, pois as válvulas eletrônicas foram substituídas por transistores e os fios de ligação por circuitos impressos.

Os de Terceira geração (1964-1970), ficaram ainda mais compactos, baratos e rápidos, pois foram construídos com circuitos integrados. Foi nessa época que foram criados os chips e começaram a ser utilizados os computadores pessoais

Na Quarta geração (1970-até os dias atuais), os computadores ficaram ainda mais pequenos, rápidos e confiáveis, pois a tecnologia foi aperfeiçoada. Surgiram os microprocessadores e os softwares integrados.

Alguns estudiosos indicam que a quinta geração já teria começado, mas não há consenso. As características desta geração seriam processadores com mais núcleos, maior potência e busca por reduzir o consumo de energia.

A quinta geração também se caracteriza pela combinação de inovações tecnológicas com a inteligência humana

Como surgiu a internet

(voltar)

A internet surgiu nos Estados Unidos, na década de 1960, durante a Guerra Fria. O seu desenvolvimento foi um processo longo e teve como objetivo criar uma rede de comunicação entre computadores do Departamento de Defesa e comunidades científicas.

A internet teve origem com a criação da Arpanet, em 1969, pela Darpa. A Arpanet era uma rede que interligava laboratórios de pesquisa e instalações militares, com o Pentágono. O seu objetivo era facilitar a comunicação e o compartilhamento de informações, além de ser resistente a ataques nucleares.

A primeira mensagem enviada pela internet foi entre computadores da UCLA e da SRI, duas instituições universitárias dos Estados Unidos.

A internet evoluiu significativamente nos anos 1970 e 1980, com o desenvolvimento de protocolos como o TCP/IP. Nos anos 1990, Tim Berners-Lee criou a World Wide Web, que tornou a internet mais acessível e amigável ao usuário.

A internet chegou ao Brasil em 1988 e foi liberada para uso privado e comercial em 1994. Atualmente, é o principal meio de comunicação do planeta, com mais de 5 bilhões de usuários.

O que são Clients?

(voltar)

No contexto do HTML, clients são os computadores que solicitam recursos e serviços a um servidor, formando a arquitetura cliente-servidor.

No desenvolvimento web, os termos "lado do cliente" e "lado do servidor" descrevem onde o código do aplicativo é executado. O lado do cliente, ou client-side, é o computador do usuário que acessa o site por meio do navegador. Já o lado do servidor, ou server-side, é o servidor que hospeda o site.

O que são Serves?

(voltar)

Server-side, ou "lado do servidor", é um termo que se refere a operações que são feitas no servidor, e não no cliente, em um contexto cliente-servidor

No HTML, o server-side pode se referir a: Server Side Includes (SSI) Um código que é incluído em uma página HTML e interpretado pelo servidor após a página ser enviada ao navegador. Os comandos SSI podem incluir informações como a data da última alteração, outro arquivo HTML, ou aplicar scripts.

Server Side Rendering (SSR) Uma técnica que consiste em renderizar páginas no servidor antes de enviá-las ao cliente. O SSR pode melhorar o desempenho e o SEO de aplicações web.

Sobre linguagens de programação web

(voltar)

Linguagem de Programação é uma maneira de escrever um conjunto de instruções e regras para gerar programas (software).

Uma linguagem de programação pode fazer parte de diferentes classificações, mas existem algumas que são essenciais para qualquer programador ou profissional de tecnologia. Segundo um levantamento realizado pela consultoria de programação norte-americana RedMonk, as linguagens de programação mais populares são JavaScript, Python, Java, PHP, C#.

Mas atenção, o HTML não é uma linguagem de programação!

HTML é um arquivo de marcação, onde os textos são delimitados por nomes que o navegador consegue interpretar e cada um destes delimitadores, tem uma função e comportamento específico.

Estrutura básica do HTML

(voltar)

A estrutura básica de um documento HTML é composta por três tags:

  • html: Define o início e o fim do documento
  • head: Contém informações sobre o documento, como o título, links para estilos e scripts
  • body: É onde fica o conteúdo principal da página, como textos, imagens e formulários

Falando sobre Tags

(voltar)

Entender sobre as Tags HTML é importante, pois elas são marcadores usados para delimitar seções de um documento HTML.

Cada tag HTML tem um significado semântico e define um bloco de conteúdo que pode ser manipulado pelo navegador. Por exemplo, as tags podem delimitar um parágrafo, texto em negrito, texto itálico e assim vai.

O uso das tags HTML são extremamente úteis porque permitem que os navegadores interpretem e apresentem corretamente o conteúdo. Além disso, as tags HTML também podem ser usadas para aplicar estilos a um documento HTML usando folhas de estilo em cascata (CSS).

Atributos básicos

(voltar)

Os atributos das tags no HTML são características associadas a um elemento, que são posicionadas dentro da tag de abertura. Cada atributo recebe um valor para configurar a característica que será dada ao elemento.

Alguns exemplos de atributos são:

  • refh: Especifica a URL da página para a qual o link vai
  • align: Alinha o conteúdo de um elemento
  • width: Define a largura do elemento

Textos

(voltar)

A tipografia HTML é o uso de fontes web em um site para garantir que o texto seja consistente em todos os dispositivos. Uma fonte HTML é importante para o design de um site, pois garante que a fonte escolhida seja renderizada corretamente em qualquer navegador e dispositivo.

A tipografia é o estudo, criação e aplicação dos caracteres, estilos, formatos e arranjos visuais das palavras.

Para estilizar uma fonte no HTML, é possível usar o CSS ou um elemento do estilo HTML. Algumas das alterações que podem ser feitas são: Mudar a cor do texto, Mudar a cor do fundo, Alterar o tamanho da fonte, Alterar o estilo ou o peso da fonte.

Um exemplo de tag usada para formatar textos é o h1

  • A tag h1 é uma etiqueta HTML que serve para marcar o título principal de uma página da web. É a heading tag mais importante e deve ser única.

Listas ordenadas e não ordenadas

(voltar)

A principal diferença entre listas ordenadas e não ordenadas no HTML é a forma como os itens são:

  • Listas ordenadas
    Os itens são enumerados, com uma contagem que pode ser feita por números, letras, algarismos romanos ou símbolos. São delimitadas pela etiqueta ol(Ordered List).
  • Listas não ordenadas
    Os itens são apresentados com marcadores antes de cada um, como pequenos círculos pretos. São delimitadas pela etiqueta ul(Unordered List).

Links

(voltar)

Links HTML são hiperlinks que conectam páginas da web, permitindo que o usuário navegue entre elas. Eles podem ser inseridos em texto ou imagens, e podem ser configurados para abrir em uma nova página ou na mesma página.

Eles são importantes para a interconectividade da web e para a experiência do usuário. Eles são uma das principais características da web, permitindo que os usuários vinculem documentos a outros documentos ou recursos.

  • Os links podem ser configurados para ter uma formatação diferente da padrão, que é azul e sublinhado.
  • O atributo href indica a relação entre as páginas para os mecanismos de pesquisa. O href é composto pela URL e pelo texto ou objeto clicável, conhecido como texto âncora, por isso, é representado pela tag a.
  • Existem links relativos e links absolutos. Os links relativos só funcionam quando as duas páginas estão na mesma unidade, enquanto os links absolutos usam o protocolo do site, mesmo que as páginas sejam diferentes.
